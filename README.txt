# Курсовой проект: Телеграм-бот "Cource-bot"

1. Описание проекта
    Этот курсовой проект представляет собой простого телеграм-бота, который выполняет определенные функции,
    на основе полученных знаний.

2. Функциональность
    Бот может выполнять следующие команды:
      - /start: Начать взаимодействие с ботом.
      - /help: Получить справку о доступных командах.
      - /погода <город>: Получить текущую погоду для указанного города.
      - /новости: Получить последние новости.


3. Установка
    1.1 Клонируйте репозиторий
        git clone https://github.com/i-a-terekhov/Cource-bot.git
        cd mybot

    1.2 Установите зависимости
        pip install -r requirements.txt  # Сохранение зависимостей: pip freeze -local > requirements.txt

    1.3 Создайте файл config.py и укажите в нем следующие параметры
        API_KEY = "your_telegram_bot_api_key"
        WEATHER_API_KEY = "your_openweathermap_api_key"

        Замените your_telegram_bot_api_key на API-ключ вашего телеграм-бота
        и your_openweathermap_api_key на API-ключ OpenWeatherMap.

    1.4 Запустите бота
        python bot.py

4. Использование
    Откройте Telegram и найдите бота "MyBot" по имени или перейдите по ссылке: t.me/MyBot.
    Начните взаимодействие с ботом, отправив команду /start.
    Используйте команды /погода и /новости, чтобы получить информацию.
